宝可梦立体图鉴

Pokemon 3D Pokedex

✨ 应用简介

宝可梦立体图鉴是一款精美的 iOS 应用，为您呈现第一到第三世代所有宝可梦的详细信息。 无论是普通形态、地区形态（阿罗拉、伽勒尔），还是超进化、超极巨化和原始回归形态， 这里应有尽有。每个宝可梦都配有生动的 3D 动画 GIF 和精美的官方立绘。

🎮 核心功能

📚完整图鉴

收录第一到第三世代（#1-#386）所有宝可梦，包括传说中的固拉多、盖欧卡和烈空坐。

🌟多种形态

支持地区形态、超进化、超极巨化和原始回归，每个宝可梦的不同形态都能轻松查看。

🎬3D 动画

每个宝可梦都有精美的 3D 动画 GIF，让您全方位欣赏宝可梦的魅力。

📊详细信息

详细的属性、种族值、描述等信息，助您全面了解每个宝可梦。

💾智能缓存

自动缓存所有图片和动画，离线也能流畅浏览，节省流量。

🎨精美界面

现代化的 UI 设计，流畅的交互体验，让浏览成为一种享受。

🔧 技术特性

Swift UIKit SceneKit URLSession NSCache FileManager ImageIO PokeAPI

采用原生 iOS 开发，使用 UIKit 构建界面，SceneKit 展示 3D 模型。 实现了多级缓存系统（内存缓存 + 磁盘缓存），确保快速加载和离线使用。 集成了 PokeAPI 获取宝可梦数据，Project Pokemon 提供 3D 模型资源。
